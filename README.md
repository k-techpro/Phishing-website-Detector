# 🔐 Phishing URL Detection System

A Machine Learning-based web application that detects whether a given URL is **Phishing** or **Legitimate** using feature engineering and classification models.

---

## 🚀 Features

- 🔍 Detect phishing websites in real-time
- 🤖 Trained ML model using Scikit-learn
- ⚡ FastAPI backend for high performance
- 📊 Feature engineering on URL structure
- 💻 Easy-to-use API with Swagger UI
- 📁 Modular project structure

---

## 🧠 Technologies Used

- Python 🐍
- Scikit-learn
- Pandas & NumPy
- FastAPI
- Uvicorn
- Joblib
